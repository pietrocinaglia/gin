The network will be built based on m, p, q (with p + q < 1), and z as follows:
- m is the number of (intralayer) edges with which a new node attaches to existing nodes;
- p is the probability that m new edges are added;
- q is the probability that m existing edges are rewired to a random chosen edge and the related node;
- z is the percentage of (interlayer) edges that are randomly created between two layers.
Note: the model behaves like a Barabási–Alber model with p = q = 0.

* Default amount of interlayer edges (z) is 20% for multi-layers, and 0% for one-layer.
